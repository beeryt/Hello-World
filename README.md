# Hello-World

Hello-World is a GitHub project for trying out different features within the git and GitHub ecosystems.

## Installation

Use [CMake](https://cmake.org/) to create cross-platform build files for this project

```bash
mkdir build
cmake ..
make
```

## Usage

```bash
./build/hello-world
```
